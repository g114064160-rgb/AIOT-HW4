# Testing & Verification

本專案以推論展示為主，未內建完整測試。以下為建議的驗證流程。

## 快速健康檢查
1. 安裝依賴：`pip install -r requirements.txt`
2. 確認模型存在：`ls myna_resnet50v2.h5`（或在側欄輸入其他路徑）
3. 啟動：`streamlit run app.py`，載入模型並以範例圖片或自備圖片測試。

## 手動驗證步驟
- **模型載入**：輸入模型路徑 → 「載入模型」 → 無錯誤訊息。
- **圖片上傳/範例**：上傳 jpg/png 或選範例後，能顯示縮圖。
- **推論結果**：點「開始辨識」後，應看到 Top-1 與長條圖；若模型類別與標籤數不一致，會有錯誤提示。

## 回歸測試建議
- 使用數張已知標籤的圖片，觀察 Top-1 是否符合預期（至少檢查每一類各 1-2 張）。
- 替換模型或類別清單時，重複上述流程確認無崩潰、標籤對應正確。

## 自動化（選配）
- 可撰寫簡易單元測試，檢查：
  - 讀圖與前處理尺寸為 (1, 224, 224, 3)。
  - 模型輸出長度與標籤數一致。
  - 範例圖片搜尋函式在目錄不存在時返回空清單。
